Digital Storeroom

Download Digital Storeroom (ZIP)

A digital solution for managing a shop storeroom, allowing users to efficiently track, add, and search for products.

Background

The digital storeroom was my project for advanced higher computing. It gives the user a digital representation of a physical storeroom, allowing them to manage products more efficiently. The storeroom layout was modelled after the shop where I work.

When I first started working at this shop, I found it tricky to memorise the location of the different products, so came up with this project.

Features

Displaying Storeroom Layout

The storeroom layout section reads in data from a csv file into a 2d array. When this file is modified, the user can update the display to reflect the new layout.

Storeroom Layout

Connecting to an External Database

The program connects to an external SQL database, allowing for persistent storage and retrieval of product information. The setup for this was in my opinion the hardest part of the project, as I hadn't previously done anything like it.

External Database

Adding and Deleting Products

The program allows the user to add or remove products from the database. This wasn't designed as a way to keep track of stock numbers, but more for where a product should be stored. The user can specify the barcode, producer, price, and location within the store.

Add/Delete Products

Searching for Products

The program allows for the user to search for a product type, or a specific product by its barcode. At my work, most of the PC's have a barcode scanner, so manually searching by name seemed an unnecessary addition.
